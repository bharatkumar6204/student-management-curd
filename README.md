Student Management System using Django (CRUD)
## Project Description
This is a Student Management System built using Django.
It allows users to perform CRUD operations:
- Add Student
- View Student List
- Update Student
- Delete Student

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- MySQL

## Features
- Add new students
- View student list
- Update student details
- Delete student records
- Responsive UI
